了解概念

  • 作用
    • 快速拉起一个服务，这个服务可以使一个容器或者多个容器组成
    • 编排管理容器
    • 定义和运行复杂 docker 镜像,避免使用 shell 指令创建

操作

在执行指令的目录下必须存在 docker-compose.yml 文件,指令只会对配置文件内的服务或者容器起效

docker-compose.yml 参数

  • 卷:volumes
  • 端口:ports
  • 环境变量:environment
  • 容器名称:container_name
  • 重启策略:restart
  • 镜像:image
  • 命令，将覆盖镜像文件内的 CMD:command
