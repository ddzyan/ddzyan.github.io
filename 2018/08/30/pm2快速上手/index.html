pm2快速上手

参考文档：https://www.npmjs.com/package/pm2

测试demo: https://github.com/ddzyan/node-module-example/tree/master/pm2

简介

pm2 是node进程管理工具，可以利用它来简化很多node应用管理的繁琐任务，如性能监控、自动重启、负载均衡等

常用命令

参数
  • file.js - 服务启动文件
  • json_conf - pm2 配置文件
  • app_name - 指定 pm2 服务名称
  • id - 指定 pm2 id
  • all 全部服务
  • json_conf 配置文件中的全部服务
  • –only - 只操作配置文件中的指定服务
